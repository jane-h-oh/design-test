# 목회메이트 (MokboMate) - 목회자 AI 콘텐츠 비서

## 1. 프로젝트 개요

**프로젝트명:** 목회메이트  
**프로젝트 유형:** 웹 프론트엔드 SPA (Single Page Application)  
**핵심 기능:** 목회자를 위한 AI 콘텐츠 생성 플랫폼 - 설교 연구, 콘텐츠 제작, 영적 케어  
**정체성:** 목회자 AI 콘텐츠 비서 (교회 관리 ERP ❌)  
**대상 사용자:** 목사, 전도사, 성경 연구자

---

## 2. 기술 스택

| 구분 | 기술 |
|------|------|
| 프레임워크 | React 18 + TypeScript |

## 4. 페이지 상세

### 4.1 대시보드 (홈)

**중앙:** 월간 사역 캘린더
- 월별 네비게이션
- 일별 일정 표시
- 클릭 시 상세 보기

**우측:** AI 절기 기반 사역 가이드
- 현재 절기 정보 (봄, 여름, 가을, 겨울)
- 절기별 권장 콘텐츠 주제
- 성경 구절 추천

### 4.2 설교 연구소

**기능:**
- 성경 검색 (전체 성경)
-章节 검색 (구약/신약)
- 구절 상세 보기
- 메모/주석 기능

**데이터:**
- 성경 JSON 파싱
- 검색 인덱싱

### 4.3 콘텐츠 스튜디오

**구성:**
1. **원고 입력 창** - 텍스트 에디터
2. **탭 결과물 미리보기**
   - 카드뉴스 미리보기
   - QT (Quiet Time) 템플릿
   - 설교 노트
   - SNS 게시물

### 4.4 영적 케어

**기능:**
- 묵상 일지
- 영적 성장 추적
-기도 요청 관리
